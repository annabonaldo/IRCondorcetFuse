Modelli scelti:
-I modelli scelti sono tutti Weighting models (se DFR appartengono al framework Divergence From Randomness). Settare -c come parameter value per term frequency normalization. I modelli field based e proximity richiedono un'indicizzazione diversa.

1. BB2 (DFR) = Bose-Einstein model for randomness. Parametro c di default o settabile
2. BM25 = Prob. model. No parametri, ne ha settati di default
3. DLH13 (DFR) = improved DLH hyper-geometric DFR mode. Non usa parametri. Highly recomended with QE
4. Hiemstra_LM: Hiemstra’s language model. Usa un parametro lambda di default (ma settabile)
5. IFB2 (DFR): Inverse Term Frequency model for randomness, the ratio of two Bernoulli’s processes for first normalisation, and Normalisation 2 for term frequency normalisation. Parametri di default o settabili.
6. TF_IDF: The tf*idf weighting function, where tf is given by Robertson’s tf and idf is given by the standard Sparck Jones’ idf. Parametri di default o settabili.

7. DFIC = Divergence From Independence model based on Chi-square statistics. No parametri.
8. DFIZ	= Divergence From Independence model based on Standardization (i.e., standardized distance from independence in term frequency tf). No parametri.
9. DirichletLM = Bayesian smoothing with Dirichlet Prior. No parametri.
10. InL2 = This class implements the InL2 weighting model. Parametri di default o settabili.
